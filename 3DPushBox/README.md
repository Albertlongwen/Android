#3D推箱子
-------------

更多细节请参阅我的博客:http://blog.csdn.net/mr_virus/article/details/41350221

###功能
- 实现经典的推箱子玩法
- 触摸屏幕可转换3D视角
- 虚拟键盘控制前进
- 可开启/关闭音效


###关于程序
- Java
- IDE:Eclipse
- OS:Windows 8.1 x64
- Android API:17
- other Environment:Opengl ES

###使用方法
可导入编译器查阅代码，也可直接使用.APK安装APP
